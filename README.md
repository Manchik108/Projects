# ClipFlow

> Менеджер буфера обмена с историей, поиском и AI-группировкой.

ClipFlow автоматически сохраняет всё, что вы копируете, и помогает быстро находить нужное. Работает в фоне, живёт в трее, не требует интернета.

---

## Возможности

- **История копирования** — всё, что вы скопировали, сохраняется в локальную SQLite-базу с FTS5-поиском
- **AI-группировка** — записи автоматически классифицируются: `код`, `ссылка`, `адрес`, `цитата`, `текст`
- **Мгновенный поиск** — полнотекстовый поиск по всей истории (FTS5)
- **Избранное** — отмечайте важные записи звёздочкой
- **Коллекции** — группируйте записи по проектам
- **Трей-иконка** — приложение висит в фоне, не мешает
- **Глобальная горячая клавиша** — `Ctrl+Shift+V` для показа/скрытия окна
- **Фильтр по категориям** — быстро переключайтесь между кодом, ссылками и т.д.
- **Полная приватность** — все данные хранятся локально, никаких облаков

## Установка

### Готовый установщик

Скачайте `ClipFlow_Setup.exe` из [релизов](../../releases) и запустите.

### Сборка из исходников

```shell
git clone <repo-url>
cd clipflow
pip install -r requirements.txt
python main.py
```

Сборка `.exe`:

```shell
build.bat
```

Готовый `.exe` появится в `dist/ClipFlow.exe`.

## Использование

1. Запустите ClipFlow — он появится в системном трее
2. Начните копировать текст как обычно — всё сохраняется автоматически
3. Нажмите `Ctrl+Shift+V` чтобы открыть окно истории
4. Ищите, фильтруйте по категориям, добавляйте в избранное
5. Нажмите `Esc` чтобы скрыть окно

### Горячие клавиши

| Действие | Клавиша |
|----------|---------|
| Показать/скрыть историю | `Ctrl+Shift+V` |
| Фокус на поиск | `Ctrl+F` |
| Закрыть окно | `Esc` |

## Технологии

- **Python 3.12** — язык
- **PyQt6** — графический интерфейс
- **SQLite + FTS5** — хранение и полнотекстовый поиск
- **PyInstaller** — сборка в `.exe`
- **NSIS** — установщик
